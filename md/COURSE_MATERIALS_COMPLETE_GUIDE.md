# 🎯 Complete Guide: Adding Course Materials to Lessons

## ✅ **Course Materials System is Ready!**

Your lesson creation and editing system now supports a comprehensive range of course materials. Here's everything you can add to your lessons:

- **Learning outcomes** and instructions
- **HTML content** support

### **2. Video Content** 🎥
- **YouTube, Vimeo, and other video URLs**

### **5. File Uploads** 📎

- **Download links** for students

### **6. Embedded Content** 🔗
- **External content** embedding
- **Custom titles** and descriptions
- **Iframe support**

### **7. Slideshows** 📊
- **External slideshow** links
- **Presentation** integration
- **Custom titles**

### **8. Quizzes** ❓
- **Quiz integration** with existing quiz system
- **Quiz ID** linking
- **Direct quiz** access

### **9. Assignments** 📋
- **Assignment integration** with existing assignment system
- **Assignment ID** linking
- **Direct assignment** access

### **10. Audio/Podcast Content** 🎵
- **Audio file uploads** (MP3, WAV, OGG, M4A)
- **Interactive audio player** with controls
- **Playback speed control** (0.5x to 2x)
- **Volume control**
- **Transcript support** for accessibility
- **Download option**
- **Perfect for**: Language learning, lectures, interviews, music lessons

### **11. Interactive Video** 🎬
- **Video with embedded questions** at checkpoints
- **Automatic pause** at specified timestamps
- **Question types**: Multiple choice, True/False, Short answer
- **Immediate feedback** after answering
- **Progress tracking** through checkpoints
- **Works with uploaded videos** (best experience) or YouTube/Vimeo URLs
- **Perfect for**: Step-by-step tutorials, case studies, safety training, knowledge checks

### **12. Code Sandbox/Interactive Code** 💻
- **Live code editor** with syntax highlighting
- **Multiple programming languages**: JavaScript, TypeScript, HTML/CSS, Python, Java, C++, SQL, JSON
- **Code execution**: Run JavaScript and HTML/CSS directly in browser
- **Real-time output**: See results immediately
- **Code templates**: Pre-filled starter code for each language
- **Read-only mode**: Option to display code without editing
- **Instructions field**: Add learning objectives or exercise instructions
- **Perfect for**: Programming courses, technical training, coding exercises, algorithm demonstrations

## 🛠️ **How to Add Materials**

### **Step 1: Edit a Lesson**
1. **Go to** your course detail page
2. **Click** on a lesson
3. **Click** "Edit Lesson" (if you're an instructor)

### **Step 2: Use the Content Builder**
1. **Scroll down** to the "Content Builder" section
2. **Click** the "Add Material..." dropdown
3. **Select** the type of material you want to add

### **Step 3: Configure the Material**
- **Text**: Use the rich text editor
- **Video**: Enter video URL and title
- **Interactive Video**: Upload video file, then add checkpoints with questions at specific timestamps
- **Audio**: Upload audio file and optionally add transcript
- **Code Sandbox**: Select programming language, enter initial code, add instructions, and optionally enable read-only mode
- **Images/PDFs**: Upload files directly
- **Quizzes/Assignments**: Enter IDs or create new ones
- **Embed**: Enter embed URL and title

### **Step 4: Save and View**
1. **Click** "Save" to save your changes
2. **View** the lesson to see how materials appear to students
