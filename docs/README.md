# Virtual Campus Documentation

## Active documentation

Active project documentation lives at the repo root and inside relevant feature
directories. There is currently no curated documentation set in this folder.

## `archive/`

The `archive/` directory contains 127 markdown files that accumulated as
point-in-time implementation notes, debugging logs, and feature summaries
during early development. They are kept for historical reference only and are
not actively maintained.

Do not rely on archived docs as a source of truth — read the code or check
the relevant CLAUDE.md / commit history instead. Archived files may reference
APIs, schemas, or features that have since changed or been removed.
